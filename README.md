# Pratica-2-Web

## Instruções para execução

O backend e frontend da página foram desenvolvidos de forma separada.

Antes de iniciar a página, primeiro inicie o backend:

```bash
# Instalar dependências
cd backend/ && npm install

# Iniciar o backend, rodará na porta 3000
npm start
```

Em outro terminal, iniciar o frontend:

```bash
# Instalar dependências
cd frontend/ && npm install

# Iniciar o frontend
npm run dev
```

O site poderá ser acessado em http://localhost:5173
